Fruit Ripeness Detection System
(Ahaar Scanner)

Intelligent sensing system to detect ripeness and quality of fruits and vegetables using spectral sensing technology. Built a full hardware-software pipeline: optics design → hardware validation → data collection → ML training → on-device inference on the ESP32-S3. Normalization constant: 30,600. Sensing via AS7341 & AS7265x.
